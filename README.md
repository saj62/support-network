# Geriatrics Support Network

A responsive website developed for Geriatrics Support Network, a nonprofit organization that provides educational resources, senior activity ideas, and expert guidance for older adults, caregivers, and families.

## Features

- Educational articles and expert resources
- Senior activity guides with downloadable materials
- Responsive layouts for desktop and mobile
- Dynamic article and activity pages
- Search engine metadata and sitemap support

## Built With

- React
- Vite
- React Router
- Tailwind CSS
- Vercel

## Live Website

[geriatricssupportnetwork.org](https://geriatricssupportnetwork.org)

This website was designed and developed as a client project for Geriatrics Support Network.
